Accounts and documents that have a filing fee

There is a fee to file accounts for:

  • overseas companies
  • community interest companies (CIC)

You should check the documents you need to file. If there is a filing fee, it will say on the document itself.

If you need to file electronic charge (mortgage) documents, you will need to apply to register as a lender instead.
