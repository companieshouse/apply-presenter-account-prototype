Use this service to apply for a presenter account to file online with Companies House using third-party software.

You can only use a presenter account to file documents without a fee. You will need to complete an application form if you need to apply:

  • for a presenter credit account to file documents with a fee
  • to register as a lender to file electronic charge documents

It takes around 5 minutes to apply. You'll need to sign in or create a Companies House account to use this service.
