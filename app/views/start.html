Use this service to apply for a presenter account to file online with Companies House using third-party software.

If you’ve lost your presenter account details

You'll need to use this service to apply for a new account.

You must contact us to report if you think someone else knows, or has access to your presenter account details.

You can only use a presenter account to file documents without a fee. You cannot use this service to:

  • apply for a presenter credit account to file documents with a fee
  • register as a lender to file electronic charge documents

For these, you will need to complete an application form.

It takes around 5 minutes to apply. You'll need to sign in or create a Companies House account to use this service.
